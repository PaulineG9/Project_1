# **Personal Website**

This is a simple flask website.


## **Prerequisites**

Requires python3, flask,venv and sqlite

## **Getting Started**

Activate the python virtual environment: *$source venv/bin/acivate*

To run the app: *$flask run*

To stop the app : *$^c*

To exit the environment : *$deactivate*
